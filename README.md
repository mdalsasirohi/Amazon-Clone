# 🛒 Amazon Clone

A frontend clone of the Amazon website built using **HTML and CSS**. This project recreates the basic layout and visual appearance of an e-commerce website as part of my web development learning journey.

## ✨ Features

* Amazon-inspired navigation bar
* Product categories and product sections
* Search bar UI
* Product cards and banners
* Clean and structured webpage design

## 🛠️ Technologies Used

* HTML5
* CSS3

## 📂 Project Structure

amazon-clone/
│
├── index.html
├── style.css
└── images/

## 🚀 How to Run

1. Clone this repository:

   ```bash
   git clone YOUR_REPOSITORY_URL
   ```
2. Open the project folder.
3. Open `index.html` in your web browser.

## 🎯 Learning Outcome

Through this project, I practiced creating webpage layouts using HTML and CSS, working with styling, positioning, and designing an e-commerce-inspired interface.
